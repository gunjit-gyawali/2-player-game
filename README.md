# 2 Player Puzzle Game

I made this small puzzle game using Godot. It’s meant for 2 players on the same keyboard. The main idea is simple: you and another person have to work together to get through each level. You can't complete the levels alone.

## What this game is

In this game, I made it so each player controls a different character. You can’t really finish levels alone, so you have to cooperate the whole time. Some puzzles need both players to do things at the same time.

I made this game in godot to expand my game dev skill.

## Controls

**Player 2**
- WASD to move  

**Player 1**
- Arrow keys to move  

## How to play

I designed the levels so you have to help each other.  
Sometimes one player holds a switch while the other moves forward.  
Just communicate and figure it out together.
